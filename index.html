PDF Generator

This project is a clean, functional web application that generates well-formatted PDF reports based on user input. Users fill out a simple form with fields for author, year, product, brand, and a description, which is then processed into a standardized PDF document. It features an intuitive UI, supports custom text content, and is ideal for creating product summaries, research reports, or documentation. The system is extendable for database integration and is built using HTML, CSS, JavaScript, PHP, and TCPDF.

Task Tracker

Task Tracker is a productivity tool designed to help individuals and teams manage daily tasks, prioritize work, set deadlines, and collaborate efficiently. It supports creating, updating, categorizing, and assigning tasks with priority levels and reminders. Users can track progress through statuses like To-Do, In Progress, and Done, organize with tags, and filter or search tasks easily. Features include a visual dashboard (like a Kanban board), real-time notifications, and cross-platform access across web, mobile, or desktop. Built using HTML, CSS, and JavaScript, it streamlines workflow and enhances time management.
